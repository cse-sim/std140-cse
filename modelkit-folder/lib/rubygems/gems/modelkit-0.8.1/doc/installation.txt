
Manual installation procedure for Params for Windows:


1. Unzip Params.zip and move the Params directory to a location of your choosing.

For example: C:\Program Files\Params


2. Edit your "Path" environment variable to append the path to the 'Params\bin' directory with a semicolon.

For example, if your original "Path" is something like this: C:\Program Files\ABC;C:\Program Files\XYZ
Then you should append to it like this: C:\Program Files\ABC;C:\Program Files\XYZ;C:\Program Files\Params\bin


3. Test the "Path" by opening a command prompt window and typing 'params' at the prompt. If you get a command-line help summary, the installation was successful.
